# 공유
- 목적: 상세 분석 링크를 OS 공유 시트나 클립보드 복사로 전달하게 한다.
- 진입/화면: `frontend/trend.html` 상세 상단의 `공유` 버튼. 저장과 후기는 로그인 없이 로컬 보관함에 남긴다.
- 표시 데이터: 공유 제목은 `한끗`, 본문은 트렌드 제목과 유형별 요약(신뢰분석은 광고/신뢰 점수, 트렌드는 `stage`) 및 `label`, URL은 현재 상세 페이지 주소다.
- 현재 상태: 구현됨. `H.share()`가 `navigator.share()`를 우선 사용하고, 미지원 환경에서는 본문+URL을 클립보드에 복사한다.
- 관련 코드: `frontend/trend.html`의 `#shareBtn` 핸들러, `frontend/assets/app.js`의 `H.share()`.
- 비고: 카카오톡 전용 공유와 이미지 카드 저장은 2차 기능이다. 공유 실패/미지원 시 확정 판정처럼 보이는 문구를 만들지 않는다.
